Modals are hard

An optimal modal should:

  1. Fit the viewport without scrolling the modal itself, only content
  2. Adjust height dynamically in mobile when keyboard is open, without overflowing
  3. Have clickable overlay background that closes the modal
  4. Have overlay work on the empty space above & under modal as well

I've put here some examples to showcase the different ways of doing it.

  • Flowbite has a pretty good scrolling modal
  • melt-ui's example is no good
  • Native dialog is too difficult to work with
  • V1 doesn't scroll content
  • V2 doesn't close on overlay clicks above/under
  • V3 is the best way I've found so far. No clickable backdrop though
